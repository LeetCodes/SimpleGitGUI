How to resolve conflicts?

Files with these status (remote/local) are conflicting:

  • Deleted/Deleted
  • Created/Unmerged
  • Unmerged/Deleted
  • Unmerged/Created
  • Deleted/Updated
  • Created/Created
  • Unmerged/Unmerged

Conflicts will be highlighted in conflicting files, they will look something like this: 

<<<<<<< HEAD
Your code (not pushed)
=======
Someone else's code (already pushed)
>>>>>>> d9927045d536195f41f6050e69c88d692e8f3da5

Go through each conflicting file and fix those conflicts.
